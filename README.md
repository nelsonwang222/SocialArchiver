# Social Link Archiver

A React application for archiving social links.

## Getting Started

### Installation

```bash
npm install
```


### Local Development

```bash
npm run dev
```

### Build

```bash
npm run build
```



## Deployment

This project is configured to deploy to **GitHub Pages** automatically via GitHub Actions.

### Prerequisites

1. Go to your repository **Settings**.
2. Navigate to **Pages** in the sidebar.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.

Once configured, any push to the `main` branch will trigger a deployment.

View your app in AI Studio: https://ai.studio/apps/drive/14c6DMbwmGKFG_-Lq0luyskLBryW4g6br

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
